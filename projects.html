Secure Aircraft Data Load

Description

Partnered with Collins Aerospace to develop a secure test environment for aircraft data loading systems. The project focused on proving the feasibility of secure communication between avionics devices when delivering critical payload data. In aviation, data loading operations are essential for updating flight software, navigation databases, and system configurations - making security and efficiency paramount.

Built a proof-of-concept system using dual Raspberry Pi devices to simulate aircraft data loading scenarios. Implemented and tested multiple encryption algorithms (AES-256, RSA) to evaluate performance versus security tradeoffs. The test environment validated secure data transfer protocols while maintaining the speed requirements necessary for real-world aviation operations.

My Role

Served as the Security Implementation Lead, responsible for designing the encryption architecture and developing the secure communication protocols. Led the hardware setup and configuration of the Raspberry Pi test environment. Conducted security analysis and penetration testing on the prototype to identify vulnerabilities before final delivery.

Skills & Knowledge Gained

  • Embedded systems security and hardware-level encryption implementation
  • Cryptographic algorithm selection and performance optimization
  • Secure communication protocols in resource-constrained environments
  • Professional collaboration with industry partner (Collins Aerospace)
  • Technical documentation and presentation to stakeholders
  • Real-world application of cybersecurity principles in aviation sector

Big Picture Contribution

This project addresses a critical need in modern aviation security. As aircraft become more connected and reliant on software updates, securing the data loading process is essential to preventing unauthorized access or malicious payload injection. Our work demonstrated that strong encryption can be implemented without sacrificing the operational efficiency required in aviation environments, potentially influencing future industry standards for secure aircraft data management.

R8MyApartment - Student Housing Platform

Description

Developed a web application to help college students find housing and roommates while providing a platform to rate and review apartment complexes. The app aggregates student reviews, rental information, and roommate matching functionality into a single platform, addressing the common pain point of finding reliable housing information and compatible roommates.

Built the rating and review system that allows students to submit and view apartment ratings based on multiple criteria (location, maintenance, price, amenities). Implemented the roommate matching algorithm based on user preferences and compatibility factors.

Skills & Knowledge Gained

Cybersecurity Network Security

Network Honeypot System

Description

Created a custom honeypot system in Python to detect and analyze malicious network activity. The honeypot simulates vulnerable services (SSH, HTTP, FTP) to attract attackers and log their tactics, techniques, and procedures. Collected data on common attack patterns, brute force attempts, and exploit payloads to better understand attacker behavior and improve defensive strategies.

My Role

Solo developer responsible for the entire implementation. Designed the honeypot architecture, implemented service emulation for multiple protocols, and built the logging and analysis system. Deployed the honeypot on a cloud server and monitored attack activity over several weeks, generating reports on findings.

Skills & Knowledge Gained

  • Python socket programming and network protocol implementation
  • Understanding of common attack vectors and exploitation techniques
  • Log analysis and threat intelligence gathering
  • Linux system administration and firewall configuration
  • Secure deployment practices for security research environments
  • Data visualization for security metrics and attack patterns

Resources Used

Python 3, socket library, asyncio for concurrent connections, SQLite for logging, iptables for network filtering, DigitalOcean for cloud deployment, Matplotlib for data visualization

Computer Architecture Hardware Design

MIPS Processor Implementation

Description

Designed and implemented a functional MIPS (Microprocessor without Interlocked Pipeline Stages) processor as part of a computer architecture course. The processor supports core MIPS instructions including arithmetic operations, memory access, and control flow. Built using hardware description language and verified through simulation and testing with various assembly programs.

My Role

Individual project where I implemented the complete datapath and control unit. Designed the ALU (Arithmetic Logic Unit), register file, instruction memory, and data memory components. Debugged timing issues and verified correct instruction execution through comprehensive test programs written in MIPS assembly.

Skills & Knowledge Gained

  • Computer architecture fundamentals and processor design
  • Hardware description language (Verilog/VHDL) programming
  • Understanding of instruction set architecture (ISA)
  • Digital logic design and finite state machines
  • Assembly language programming and machine code
  • Hardware simulation and debugging techniques

Resources Used

Verilog HDL, ModelSim for simulation, MIPS assembly language, Quartus Prime for synthesis, FPGA development board for hardware testing (optional deployment)
